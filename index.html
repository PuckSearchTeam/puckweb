Puck is an architecture refactoring tool designed to enhance your development experience by providing bad dependencies identification within IntelliJ. It dynamically generates and exhibits within the code all violations of the coupling constraints defined using Weland language.

Weland is a language interpreted by Puck to express coupling constraints. You can specify for each element of your program by whom it is accessible. You can easily browse through the constraint violations using filters based on violations sources or targets.
