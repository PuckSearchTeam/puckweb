Welcome to the Puck Plugin page! Here you can find the latest downloads, updates, and instructions for using the Puck Plugin with IntelliJ.

Puck Plugin is designed to enhance your development experience by providing robust search and navigation capabilities within IntelliJ.

Features include:

  • Advanced code search
  • Contextual navigation
  • Integration with version control systems

Downloads

Download the latest version of the Puck IntelliJ plugin here.

You can then install the plugin as follows:

  • Open IntelliJ
  • Select "Plugins"
  • Click on the toothed wheel
  • Select "Install Plugin from Disk..."
